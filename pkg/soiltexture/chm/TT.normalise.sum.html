TT.normalise.sum(soiltexture)R Documentation

Normalises the sum of the 3 particle size classes.

Description

Normalises the sum of the 3 particle size classes in tri.data to text.sum (100%).

Usage

Arguments

tri.data
css.names
text.sum
text.tol
tri.pos.tst
residuals
